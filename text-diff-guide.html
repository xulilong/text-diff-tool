文本对比工具怎么用

文本对比工具适合比较两段代码、配置文件、接口返回、合同文案或 Markdown 文档，帮助你快速看到新增、删除和修改内容。

基本使用步骤

  1. 把旧版本内容粘贴到左侧，把新版本内容粘贴到右侧。
  2. 选择文本类型，或者保持自动识别。
  3. 需要时先点击格式化，例如 JSON、HTML、XML 或 SQL。
  4. 点击开始对比，查看差异统计和详细对比结果。
  5. 如果需要留档，可以导出对比结果。

适合对比哪些内容

  • 代码修改：查看函数、配置项、样式和 SQL 的变化。
  • JSON 数据：比较接口返回、测试数据或配置文件差异。
  • 文案版本：比较合同条款、公告、产品说明和运营文案。
  • 日志片段：定位两次运行结果中不同的字段或错误信息。

什么时候需要先格式化

如果内容是一整行 JSON、压缩 HTML 或没有缩进的 SQL，先格式化再对比通常更容易阅读。格式化后，对比结果会按更清晰的行结构展示。

隐私与本地历史

文本对比主要在浏览器本地完成。历史记录功能使用浏览器本地存储，适合保存常用对比片段，但处理敏感文本后建议及时清理。
